# discourse-anonymous-post

A Discourse plugin for posting as anonymous user via a checkbox in composer, without the need to switch to anonymous mode.

## Известные ограничения

Email-уведомления, mailing-list mode и дайджесты пока не анонимизируются этим плагином. Это известный открытый канал: реальные авторы могут попадать в тему или тело письма. Исправление email-канала вынесено в отдельный этап.
